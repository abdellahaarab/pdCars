# 🚗 PD Cars - Premium Cars Web App

PD Cars is a modern automotive web interface for exploring premium cars, brands, articles, reviews, and news. Built with **HTML + Tailwind CSS + Vanilla JavaScript**, and designed to be fully compatible with future external APIs.

---

## ✨ Features

- 🚘 Browse premium cars
- 🏷️ Explore car brands
- 📰 Read automotive articles
- ⭐ View car reviews
- 🔎 Search & filter system
- ❤️ Favorites (LocalStorage)
- ⚖️ Car comparison tool
- 📱 Fully responsive design
- ⚡ Fast and lightweight

---

## 📁 Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- JSON Database (Local first)
- API-ready architecture

---

## 📦 Project Structure

- `/index.html` → Home page  
- `/cars.html` → Cars listing  
- `/articles.html` → Blog system  
- `/brands.html` → Car brands  
- `/reviews.html` → Reviews system  
- `/data/` → JSON database files  
- `/assets/` → CSS, JS, images  

---

## 🚀 Future Upgrade

This project is designed to easily connect with external APIs in the future without changing the frontend logic.
